# Workspace (Replit / dev)

pnpm monorepo — TypeScript, Express 5, PostgreSQL + Drizzle, Vite React apps.

## Commands

- `pnpm run typecheck` — all packages  
- `pnpm run build` — typecheck + build  
- `pnpm --filter @workspace/api-server run dev` — API  
- `pnpm --filter @workspace/bridge-design-suite run dev` — web UI
