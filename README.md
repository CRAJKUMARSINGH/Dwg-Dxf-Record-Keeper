# BridgeMaster Pro 2026

**The Professional Bridge Engineering Software Suite for India**

BridgeMaster Pro is a high-performance parametric CAD generator and design suite tailored for bridge engineers, consultants, and government departments (PWD / NHAI / Railways).

---

## 🚀 Key Modules

### 1. Drawing Generator
Generate high-fidelity, audit-ready drawings in seconds:
- **GAD & Plan Views**
- **Longitudinal & Cross Sections**
- **Substructure Details** (Pier, Abutment, Wing Wall)
- **Foundation Layouts** (Open, Raft)
- **Box Culvert & T-Beam Templates**

### 2. Quantity (BOQ) Module
Instant, accurate volume calculations:
- Concrete (Deck, Pier, Abutment, Found)
- Reinforcement Steel (MT)
- Earthwork & Excavation
- **Export to Formatted Excel / PDF**

### 3. Design Auditor
Real-time engineering validation:
- **IRC:112** Span-to-depth checks
- **IRC:SP:13** Vertical clearance (Freeboard) validation
- Minimum thickness & road width alerts

### 4. Smart Terrain Engine
Automated profiling using NSL data:
- Linear terrain interpolation
- Auto-calculated pier/abutment heights based on ground slope
- 2D viewport for live parameter updates

## 💻 Getting Started

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch the desktop application:
   ```bash
   python main_app.py
   ```

## 🛠️ Tech Stack
- **UI**: PySide6 (Qt)
- **CAD**: ezdxf
- **Visualization**: Matplotlib
- **Reporting**: ReportLab (PDF), xlsxwriter (Excel)
- **Data**: Pydantic (Validation), JSON (Persistence)
